¿Como funciona el programa? 

-Con un límite de 2000 mb por archivo podras subir una base de datos, misma que podras graficar en automatico con librerias como pandas en python. El analisis es en automatico
por consecuente es bastante recomendable que las bases de datos que deseas subir sean previamente limpiadas y filtradas para que a la hora de graficar no tengas errores. 
Considera que es un programa limitado por entendido de que las bases de datos a graficar deben de ser simples ya que no es un proyecto para bases de datos grandes. Aunque he de 
decir que el apartado de IA sí puedes subir bases sin filtrar o limpiar ya que detecta todo sin ningun inconveniente.
